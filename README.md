# 2D Beginner: Adventure Game

Unity project for the Unity Learn course `2D Beginner: Adventure Game` using Unity 2022.3 LTS.

## Course coverage

- Player character and movement: 2D player, keyboard input, input-action style code, Rigidbody2D movement, frame-rate independent fixed update.
- Game environment and physics: generated tilemap world, decorative sprites, Rigidbody2D collision, tilemap collision through composite colliders.
- Health system: max/current health, health pickups, full-health pickup guard, static hazards, invincibility window.
- Heads-up UI display: UI Toolkit HUD with health bar, health text, objective text, and dialogue panel.
- Characters and interaction mechanics: patrolling malfunctioning enemies, animator parameters, repair projectiles, enemy repair state, NPC dialogue through raycasting.
- Enhance and polish: follow camera, audio sources and generated SFX/music, repair particles, pickup particles, and build settings with the main scene.

## Controls

- Move: `WASD` or arrow keys.
- Fire repair projectile: `Space`.
- Talk/interact: `E`.

## Generated content

The scene, sprites, tiles, prefabs, animations, and build settings are generated by `Assets/Scripts/Editor/AdventureGameProjectSetup.cs`.

Run from Unity with `Tools > 2D Beginner Adventure > Rebuild Course Project`, or batchmode:

```powershell
& 'C:/Program Files/Unity/Hub/Editor/2022.3.62f3/Editor/Unity.exe' -batchmode -quit -projectPath . -executeMethod AdventureGameProjectSetup.RebuildCourseProject
```
